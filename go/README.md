# dickless.io Go Example

## SDK

This example uses [`dickless-go`](https://pkg.go.dev/github.com/dickless-io/dickless-go).

## Setup

```bash
go get github.com/dickless-io/dickless-go
```

## Run

```bash
export DICKLESS_API_KEY="dkls_live_YOUR_KEY"
go run main.go
```

## What it does

Demonstrates content moderation, PII redaction, prompt sanitization, sentiment analysis, URL shortening, input validation, and the AI Gateway.
