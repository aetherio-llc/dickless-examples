# dickless.io SDK Examples

Quickstart examples for every officially supported SDK. Each subdirectory is a self-contained project you can clone and run.

| Language | Directory | SDK |
|----------|-----------|-----|
| JavaScript / TypeScript | [`javascript/`](javascript/) | `@dickless/sdk` |
| Python | [`python/`](python/) | `dickless` |
| Go | [`go/`](go/) | `github.com/dickless-io/dickless-go` |
| Ruby | [`ruby/`](ruby/) | `dickless` |
| Rust | [`rust/`](rust/) | `dickless` |
| C# | [`csharp/`](csharp/) | `Dickless.Sdk` |
| Java | [`java/`](java/) | `io.dickless:dickless-sdk` |
| PHP | [`php/`](php/) | `dickless/sdk` |
| Swift | [`swift/`](swift/) | `dickless-swift` |
| Kotlin | [`kotlin/`](kotlin/) | `io.dickless:dickless-sdk` |
| R | [`r/`](r/) | `dickless` |

## Getting Started

1. [Sign up](https://dickless.ai) and grab an API key
2. Pick your language directory
3. Follow the README inside

## What Each Example Covers

- Content moderation
- PII redaction
- AI Gateway (OpenAI, Anthropic, Google)
- Prompt sanitization
- URL shortening
- Sentiment analysis
- And more
