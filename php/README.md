# dickless.io PHP Example

## SDK

This example uses [`dickless/sdk`](https://packagist.org/packages/dickless/sdk).

## Prerequisites

- PHP 8.1+
- Composer

## Setup

```bash
composer require dickless/sdk
```

## Run

```bash
export DICKLESS_API_KEY="dkls_live_YOUR_KEY"
php example.php
```

## What it does

Demonstrates content moderation, PII redaction, prompt sanitization, sentiment analysis, URL shortening, input validation, and the AI Gateway.
